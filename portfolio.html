Introducing Task Management System

Task Management

Description

A robust console-based Task Management System built in Java. This project demonstrates real-world object-oriented design, layered architecture, data persistence, and testing. It supports user roles, task management, and integrates a modular structure with over 30 classes.

Features

Introducing Gladiator Console Game

Gladiator

About the Game

Gladiator Game is a turn-based strategy game where you take on the role of a gladiator fighting in an arena. Your goal is to defeat opponents using attacks, aggressive attacks, and elixirs that enhance your abilities. The game is written in Java and follows object-oriented programming principles.

Features

  • Turn-based combat system

Introducing Pokemon CRUD App

Pokemon

About the Game

This is a Java console application for managing Pokemon and their trainers using a MySQL database. It uses HikariCP for connection pooling and JDBC for database operations. The app provides full CRUD functionality for both Pokemon and trainers, along with additional features like catching Pokemon, evolving them, and viewing a trainer's Pokemon.

Features

  • Create, read, update, and delete trainers and pokemons.
  • View all Pokemon or filter by type or level.
  • Catch a Pokemon and assign it to a trainer.
  • Evolve a Pokemon.
